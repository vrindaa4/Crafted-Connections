# 🧵 Crafted Connections

**Crafted Connections** is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application that serves as a digital marketplace for handmade and artisanal products. The platform connects creators with buyers who appreciate unique, handcrafted items.

---

## ✨ Features

- 🛍️ Browse and search for handmade goods
- 👩‍🎨 Seller onboarding and product listings
- 🧾 Product detail pages with images and pricing
- 🛒 Add to cart and checkout flow
- 💳 Payment gateway integration (Stripe/PayPal)
- 🧑‍💻 Secure user authentication (JWT)
- 📦 Order management and tracking
- 🔐 Admin dashboard for product & order control

## 🧰 Tech Stack

| Technology | Description |
|------------|-------------|
| MongoDB    | NoSQL Database for storing user, product, and order data |
| Express.js | Backend web framework for routing and API logic |
| React.js   | Frontend UI library with component-based structure |
| Node.js    | JavaScript runtime to run backend server |
| Mongoose   | ODM for MongoDB |
| JWT/Auth   | For secure login/signup |
| Cloudinary / Multer | Image Uploading and Storage |
| Stripe / Razorpay | Payment gateway |
| Tailwind CSS | (Optional) Utility-first CSS for clean design |
